# Russia–Ukraine War Analysis Dashboard
This project analyzes the **Russia–Ukraine War** dataset using **SQL**, **Python**, and **Streamlit** to explore military, economic, and humanitarian trends. The objective was to clean and analyze the data, uncover key insights, and present them through an interactive dashboard.

##  What I Did
* Cleaned and prepared the war dataset for analysis.
* Used SQL to explore trends and answer analytical questions.
* Analyzed military losses, equipment destruction, territorial changes, and humanitarian impacts.
* Created visualizations to compare trends over time.
* Built an interactive Streamlit dashboard for easy exploration of the data.

## Tech Stack

* SQL
* Python
* Pandas
* Matplotlib
* Streamlit
* Git & GitHub

## Key Takeaway

This project demonstrates an end-to-end data analysis workflow, from cleaning raw data and performing SQL analysis to building an interactive dashboard that highlights key trends and insights related to the Russia–Ukraine conflict.

> **Note:** This project is intended for educational and analytical purposes only. The visualizations are based on publicly available data and should not be interpreted as political commentary.
